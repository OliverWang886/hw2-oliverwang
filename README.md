# Week 2 GenAI Workflow Prototype

## Business Workflow
This project explores a simple GenAI workflow for turning meeting notes into action items.

## User
The user is a project manager, team lead, or operations coordinator who needs to review meeting notes and quickly identify next steps.

## Input
The system receives short meeting notes or meeting transcripts in plain text.

## Output
The system produces a structured list of action items, including the task, the owner if available, and the deadline if mentioned.

## Why This Task Is Valuable
This task is valuable because meeting notes are often messy, incomplete, and time-consuming to process manually. A GenAI system could save time by producing a first draft of action items that a human can review and edit.

## Prototype Status
The prototype successfully runs from the command line and makes a live Gemini API call to generate structured action items from meeting notes.
